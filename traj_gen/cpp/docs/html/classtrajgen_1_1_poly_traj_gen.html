Detailed Description

template<typename T, size_t dim>
class trajgen::PolyTrajGen< T, dim >

Piecewise polynomial trajectory. In this method, we define the primitive of trajectory as polynomial.

Optimization is performed w.r.t the coefficient of polynomials or free end-derivative of each polynomial segment

template<typename T , size_t dim>
MatrixRow< T > trajgen::PolyTrajGen< T, dim >::coeff2endDerivatives ( const spMatrixRow< T > &  Aeq)
private

This function is permutating Aeq*p = beq -> [Aeq ; Ap]*p (Afp * p) = [beq ; bp], where Ap*p = bp is a mapping from poly-coeff to the free end-derivatives. For the convenience, we consider the free end-derivative at the initial end point.
